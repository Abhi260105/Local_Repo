# To-Do Application

A simple, clean, and functional To-Do application designed to practice project structure, UI basics, and version control workflows. The project focuses on task creation, deletion, and smooth user interaction.

## Features
- Add new tasks
- Mark tasks as completed
- Delete tasks
- Persistent storage (localStorage or backend, depending on implementation)
- Minimal and responsive UI

## Tech Stack
- HTML, CSS, JavaScript
- Version control with Git
- GitHub for project hosting and collaboration

## How to Use
1. Add a task using the input field.
2. Mark tasks as complete when done.
3. Delete unwanted tasks.
4. Tasks remain saved even after page refresh (if localStorage is enabled).

## Project Status
Actively improving the UI, structure, and functionality as part of Git and GitHub learning.

## Future Improvements
- Better UI styling
- Edit existing tasks

## Purpose of the Project
This project is built to learn:
- How to structure a small front-end application
- How to manage changes with Git
